雲蝦米

Beta

特色

  • 只需要瀏覽器就可以使用的嘸蝦米輸入法！
  • 純 Javascript 實做，不需要連 Server 所以沒有安全問題(洩漏打了什麼字)！
  • 雲端自動更新！
  • 開放源始碼！歡迎來 Github 貢獻!

使用說明

  • 在任何一個頁面按下書籤就會載入雲蝦米，載入成功後會出現「等待輸入」字樣，接著就可以開始輸入了。
  • 第一次載入的時候會很慢，因為有 10MB 的資料要下載。不過接下來因為browser會有快取所以就會快很多了!
  • 按下「嘸」字或是 Ctrl + Shift 可以切換英文輸入

安裝

  1. 桌面: 請把這個連結拖到書籤列: 雲蝦米

  2. iPad Safari/手機: 複製下面的程式碼貼到書籤或網址列

    javascript:(function load() {var host="https://aitjcize.github.io/Cloud-Liu";var s1=document.createElement("script");var c1=document.createElement("link");s1.setAttribute("src",host+"static/js/cliu-loader.min.js");c1.setAttribute("href",host+"static/css/cloud-liu.css");c1.setAttribute("rel","stylesheet");document.body.appendChild(s1);document.head.appendChild(c1);})();
