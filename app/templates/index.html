雲蝦米

Beta

特色

  • 只需要瀏覽器就可以使用的嘸蝦米輸入法!
  • 純 Javascript 實做，不需要連 Server 所以沒有安全問題(比如說洩漏打了什麼字)！
  • 雲端自動更新！
  • 開放源始碼！歡迎來 Github 貢獻!

使用說明

  • 第一次載入的時候會很慢，因為有8MB的資料要下載。不過接下來因為browser會有快取所以就會快很多了!
  • 在任何一個頁面按下那個書籤就會載入雲蝦米，載入成功後會出現"等待輸入"字樣，接著就可以開始輸入了。
  • 按下"嘸"字或是 Ctrl + Shift 可以切換英文輸入

安裝

  1. 桌面: 請把這個連結拖到書籤列: 雲蝦米
